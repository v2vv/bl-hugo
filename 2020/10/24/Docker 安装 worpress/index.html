环境搭建

安装Docker

Ubuntu 14.04 16.04 (使用apt-get进行安装)，其他发行版见https://dev.aliyun.com/search.html

安装docker-compose

开始安装

构建并运行项目

运行 docker-compose up -d Compose 就会拉取镜像再创建我们所需要的镜像，然后启动 wordpress 和数据库容器。 接着浏览器访问 127.0.0.1:8000 端口就能看到 WordPress 安装界面了。
